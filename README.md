# Universidad-uk challenge

## Install and run

Install with npm

```bash
    clone git@github.com:EzequielVilla/universidad-uk-challenge.git
    cd universidad-uk
    npm install
    npm run dev
```

## Environment Variables

For simplify, I extracted the .env from the .gitignore
